The Spider Number Game is a browser-based multiplayer number prediction game developed using HTML, CSS, and JavaScript. It is a simple game where the user enters a number between 0 and 100 while such number are assigned to the bots too.Everyone starts with 3 lives. The average of these numbers is calculated and 0.8 of the average is the spider number. Whose number is close to the spider number, they are the winner. The game ends if the user's lives end or they end up as the only active player.

The application features a simple and responsive user interface, real-time result calculation, active player tracking, visible life counters, winner announcements, and game-ending conditions such as Victory and Game Over. A restart option is also included, allowing users to reset the game and play multiple sessions without reloading the page.
